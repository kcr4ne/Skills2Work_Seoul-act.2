# 디지털 세상 이해
### ICBM 기술 소개
- **I**oT(Internet of Things) : 모든 사물이 인터넷에 연결되어 데이터를 주고받는 기술

  1 - 센서 : 온도, 습도, 움직임, 소리 등을 센서를 통해 감시
  2 - 네트워크 : Wi-Fi, 블루투스, 5G 등의 네트워크를 통해 수집한 데이터를 전송
  3 - 데이터 처리 : 전송받은 데이터를 분석 및 처리
  4 - 사용자 인터페이스 : 스마트폰 앱, 웹 사이트 등으로 결과를 확인

일상 속 IoT - 스마트폰, 스마트 홈 IoT, 웨어러블 IoT, 스마트 가전
산업 분야 속 IoT - 스마트 팩토리, 스마트 시티, 스마트 농업, 헬스케어 IoT

#### 데이터는 새로운 석유다
데이터나 석유 모두 가공하지 않으면 가치 X
가공 시 엄청난 가치 창출 가능
데이터 경제의 급부상, 뛰어난경제적 가치, 데이터 관리의 중요성
#### 빅데이터의 5가지 특성 : 5V 
- Volume(크기) : 얼마나 많은가 / 데이터 규모
- Velocity(속도) : 얼마나 빠른가
- Variety(다양성) : 다양한 데이터 처리 가능
- Veracity(신뢰성) : 얼마나 믿을 만한가 / 데이터 품질 문제, 신뢰성 확보 방법
- Value(가치) : 얼마나 가치 있는가 / 가치 창출 프로세스, 비즈니스 가치

#### 빅데이터 분석 과정
수집 -> 저장 -> 처리 -> 분석 -> 활용

#### 빅데이터의 한계와 주의점

2. 분석역량 부족

#### 정보보안의 3요소(CIA)
- 기밀성(Confidentiality)
- 무결성(Integrity)
- 가용성(Availability)

#### CIA의 균형점
높은 기밀성 <-> 낮의 편의성
높은 무결성 <-> 높은 비용
높은 가용성 <-> 높은 복잡성

#### 산업별 CIA 우선 순위
1. 금융업 : 기밀성 > 무결성 > 가용성
2. 의료업 : 가용성 > 기밀성 > 무결성
3. 제조업 : 가용성 > 무결성 > 기밀성

#### 개인정보보호법 핵심 원칙
- 최소 수집, 이용 원칙
- 정확성 유지 원칙
- 목적 제한 원칙
- 안전성 확보 원칙

#### 일상에서 실천하는 개인정보보호
1. 앱 설치 전 권한 확인
2. 강력한 비밀번호 사용
3. 2단계 인증(OTP) 활성화
4. 공용 와이파이 사용 주의
5. 주변 사람과의 약속 조심

### MS SC-900
- Microsoft의 보안 분야 입문 자격증
- 4개의 주요 영역으로 이루어진 시험범위
    - 보안, 컴플라이언스, ID 개념
    - Microsoft ID 및 엑세스 관리 솔루션
    - Microsoft 보안 솔루션
    - Microsoft 컴플라이언스 솔루션
- 시험 형태 & 자격증 유효 기간
    - 40~60 문항
    - 1000 중 700 점 이상 합격점
    - 온라인 감독 방식 또는 테스트 센터에서 시험
    - 유효기간: 평생, 갱신 필요 없음

#### Microsoft 보안 자격증 학습 로드맵
1. SC-900 : 보안 입문자
2. AZ-500 : Azure 보안 엔지니어 대상
3. SC-200 : 보안 운영 분석가 대상
4. SC-300 : ID 및 액세스 관리자 대상
5. SC-400 : 정보보호 관리자 대상

#### Microsoft 보안 자격증 취득 혜택
1. 실무형 지식 습득
2. 진로 및 커리어 기반 마련
3. 국제 자격증이 주는 경쟁력

### AI 기본 개념

#### AI란 무엇인가?
**기계가 사람처럼 생각하고 행동하도록 만드는 과학과 공학 기술**
궁극적 목표 : 지각, 추론, 학습, 자율 행동

#### AI가 잘하는 일은?
- 정밀 예측
- 실시간 의사결정
- **업무 자동화**

#### AI의 실생활 예시
- 스마트폰 음성 비서
- 자율주행차
- 병원 AI 영상 판독

#### AI 분류 & 3대 축
- 기능별 분류
    - ANI(좁은AI) : 과제 특화
    - AGI(범용AI) : 사람 수준 지능(연구단계)
    - ASI(초지능) : 사람 능력 초월(미래 담론)
- 학습 방법
    - 지도학습
    - 비지도학습
    - 강화학습
- 핵심 하위 영역
    - 컴퓨터 비전(딥러닝)
    - 자연어 처리(딥러닝)
    - 음성 인식(딥러닝)
    - 로보틱스(현업AI)
    - 지식 추론(현업AI)
    - AI윤리 & 안전(현업AI)

#### AI 시스템 구성 요소
- 데이터
- 모델
- 학습
- 추론
- MLOps : 모델 관리, 데이터, 모니터링 등 관리 작업

#### 2025년 생성형 AI 성능
- 언어 LLM
- 멀티모달 이해
- 로코모션 로봇

#### AI의 태동
1. 1950년 튜링 논문 - "기계도 인간처럼 대화할 수 있는가?" -> 튜링 테스트 제안
2. 1956년 다트머스 회의 - 'Artificial Intelligence' 명명, 6주 연구 캠프로 AI 커뮤니티 탄생
3. 초기 연구 - 심볼릭 추론 Logic Theorist, 체스 프로그램, 로젠블랫 퍼셉트론 등장
4. 한계 - 컴퓨터 데이터 부족으로 성과 제약, 낙관적 예측과 현실 사이 간극 노출
